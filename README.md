# SOC Alert Workflow Simulator

A real-time dashboard simulation of a Security Operations Center (SOC) alert pipeline.

This project demonstrates how alerts move through a triage workflow while metrics and activity logs update in real time.

## Features

- Real-time alert generation
- Severity classification (LOW / MEDIUM / HIGH)
- Alert workflow pipeline
  - Queued
  - Investigating
  - Closed
- Live dashboard metrics
- Activity logging system
- Randomized alert generation
- Visual severity indicators
- Flashing HIGH priority alerts

## Technologies

- Godot Engine
- GDScript
- Event-driven simulation
- UI dashboard systems
